# Проект "Игры разума"

## Системные требования
---
node version >= 17.9.0
npm version >= 8.17.0
## Установка
---
Для установки выполните следующие команды:
```
git clone git@github.com:Kirill-Chigirev/frontend-project-lvl1.git
make install
make lint
```
